# C语言内置函数

>头文件: ctype.h

内置函数|返回值|说明
-|-|-
int isupper(int)|0假,非0真|判断是否为大写字母
int islower(int)|0假,非0真|判断是否为小写字母
int isalpha(int)|0假,非0真|判断是否为字母
int isdigit(int)|0假,非0真|判断是否为数字
int toupper(int)|0失败,1成功|转为大写字母
int tolower(int)|0失败,1成功|转为小写字母
